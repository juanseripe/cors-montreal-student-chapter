## Description

In this hands-on workshop, you will learn how to build interactive dashboards in Python using Dash, a free and open-source framework. With a focus on operations research applications, we will explore how to develop responsive visualizations, design and customize layouts, and transform OR data into clear, intuitive, and interactive interfaces.
